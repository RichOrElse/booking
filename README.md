# Booking

This is an exercise web app.

## Prerequisites

Install the following:
* Ruby
* SQLite
* Yarn

## Setup

Install Ruby gems:

    $ bundle install

Install Yarn packages:

    $ yarn install

Setup the database:

    $ rails db:migrate

Run the test:

    $ rspec --format doc

Run the server:

    $ rails server -p 3000

Then open on your browser [http://localhost:3000](http://localhost:3000)
